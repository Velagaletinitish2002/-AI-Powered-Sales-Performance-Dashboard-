## 🚀 Project Overview
This project demonstrates an end-to-end **AI-powered data analytics workflow** using Microsoft Excel. It covers data collection, cleaning, transformation (ETL), analysis, and dashboard creation for a jewelry shop sales dataset.

The goal is to convert raw sales data into meaningful insights using Excel tools like Power Query, Pivot Tables, and KPI dashboards.

---

## 🎯 Objectives
- Clean and preprocess raw data
- Handle missing and duplicate values
- Perform ETL (Extract, Transform, Load) operations
- Analyze sales performance
- Build an interactive dashboard using Excel

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Power Query Editor (ETL)
- Pivot Tables
- Charts (Line, Bar)
- KPI Cards

---

## 📂 Dataset Description
The dataset contains:
